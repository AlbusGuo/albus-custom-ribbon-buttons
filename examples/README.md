# 自定义图标测试示例

本文件夹包含了一些SVG图标示例，可用于测试自定义图标功能。

## 测试步骤

1. 在 Obsidian 中打开插件设置
2. 点击任意按钮的图标选择按钮
3. 在图标列表顶部点击 "📁 自定义图标..."
4. 选择 "上传SVG文件" 标签页
5. 点击文件选择，选择本文件夹中的一个或多个 .svg 文件
6. 点击 "添加" 按钮
7. 自定义图标应该会出现在图标选择列表中，并标记为 "(自定义)"
8. 选择自定义图标，它应该会显示在按钮上

## 测试用SVG文件

- `test-clock-icon.svg` - 时钟图标
- `test-star-icon.svg` - 星星图标  
- `test-heart-icon.svg` - 心形图标

## 手动粘贴测试

也可以测试粘贴SVG代码的功能：

1. 打开任一 .svg 文件
2. 复制其中的内容
3. 在自定义图标对话框中选择 "粘贴SVG代码" 标签页
4. 输入一个图标ID（例如：my-test-icon）
5. 粘贴SVG代码
6. 点击 "添加" 按钮

## 预期结果

- 上传成功后应显示成功通知
- 自定义图标会在图标选择列表中显示
- 自定义图标可以正常应用到按钮上
- 自定义图标会随插件设置保存
- 重新加载 Obsidian 后自定义图标仍然存在
